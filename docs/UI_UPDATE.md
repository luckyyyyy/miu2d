# UI 界面优化更新

## 更新日期
2026-01-26

## 更新内容

### 🎨 标题界面重新设计 (TitleScreen)

**主要改进：**
- ✨ 使用 Ant Design 组件库重构界面
- 🎭 添加渐变背景和动态光晕效果
- 🎬 添加淡入动画，让界面加载更优雅
- 🎯 使用专业的按钮组件，提升交互体验
- 📝 改进字体样式，使用楷体展现武侠风格
- 🌟 标题添加多层阴影和描边效果，更有立体感

**视觉特点：**
- 背景：深蓝紫渐变 + 动态光晕
- 标题：金色楷体 + 多层光晕效果
- 按钮：渐变紫色主按钮 + 半透明次要按钮
- 动画：所有元素淡入效果，错开延迟

### 🗺️ 地图查看器界面重新设计 (MapViewerScreen)

**主要改进：**
- 🎨 使用 Ant Design 专业组件
- 📦 顶部控制栏：卡片式设计，毛玻璃效果
- 🎯 下拉菜单：使用 Dropdown 组件替代原有列表
- 🏷️ 键盘提示：使用 Tag 组件展示快捷键
- 📍 底部信息：使用 Badge 显示地图序号
- 🎭 统一的深色主题，半透明背景

**交互优化：**
- 按钮分组（上一张/地图选择/下一张）
- 下拉菜单选择地图，自动高亮当前项
- 图标美化（返回、地理位置、方向等）
- 响应式卡片布局

### 🎨 全局主题配置

在 `main.tsx` 中配置了 Ant Design 主题：

```typescript
<ConfigProvider
  locale={zhCN}
  theme={{
    algorithm: theme.darkAlgorithm,
    token: {
      colorPrimary: "#667eea",    // 主题紫色
      colorSuccess: "#10b981",    // 成功绿色
      borderRadius: 8,
      fontSize: 14,
    }
  }}
>
```

**主题特点：**
- 深色模式（darkAlgorithm）
- 自定义紫色主题色
- 中文本地化（zh_CN）
- 统一的圆角和字体大小

## 界面预览

### 标题界面特点
1. **标题**：金色楷体，多层光晕和阴影
2. **副标题**：月影传说，古铜色
3. **菜单卡片**：毛玻璃效果，紫色渐变按钮
4. **版权信息**：灰色小字，居中显示
5. **背景动画**：黄色和蓝色光晕循环脉动

### 地图查看器特点
1. **顶部控制栏**：黑色半透明卡片
2. **按钮组**：返回 + 上一张 + 地图选择 + 下一张
3. **快捷键提示**：右侧灰色卡片，Tag 标签展示
4. **底部信息**：左下角卡片 + Badge 徽标
5. **下拉菜单**：点击地图名称显示所有地图

## 开发说明

### 运行项目
```bash
pnpm dev      # 开发模式
pnpm build    # 生产构建
```

### 自定义主题
如需修改主题色，编辑 `src/main.tsx` 中的 ConfigProvider 配置。

### 添加新页面
参考 TitleScreen 和 MapViewerScreen 的实现方式，使用 Ant Design 组件保持风格一致。

## 优势

1. **专业性**：使用成熟的 UI 组件库
2. **一致性**：统一的视觉语言和交互模式
3. **可维护性**：组件化开发，易于修改和扩展
4. **可访问性**：Ant Design 内置无障碍支持
5. **响应式**：自适应不同屏幕尺寸
6. **国际化**：支持中文本地化
